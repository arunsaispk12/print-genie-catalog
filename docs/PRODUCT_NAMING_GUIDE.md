# Print Genie - Product Naming Guide

## Overview
This guide establishes consistent naming conventions for all Print Genie products to ensure clarity, searchability, and professional presentation across all platforms.

---

## Naming Formula

### For Pre-Designed Products
```
[Primary Function/Type] - [Distinctive Feature] - [Material] ([Size/Variant])

Examples:
- Phone Stand - Adjustable Angle - PLA (Medium)
- Desk Organizer - 5 Compartment - PETG (Large)
- Planter - Geometric Hexagon - PLA+ (Small)
```

### For Custom Prints
```
Custom [Product Type] - [Customer Name/ID] - [Material]

Examples:
- Custom Functional Part - Order #C12345 - Nylon
- Custom Miniature - ClientABC - Resin
- Custom Prototype - ProjectXYZ - ABS
```

### For Services
```
[Service Type] - [Specification] - [Delivery Timeline]

Examples:
- Rapid Prototyping - Functional Testing - 48hr
- Iterative Development - 3 Revisions - 1 Week
- Design Consultation - CAD Review - 24hr
```

### For Materials & Filaments
```
[Material Type] - [Brand/Quality] - [Color] - [Weight]

Examples:
- PLA+ Filament - Premium - Galaxy Blue - 1kg
- PETG Filament - Standard - Clear - 500g
- TPU Flexible - Ultra Soft - Black - 1kg
```

## Naming Components

### 1. Primary Function/Type
**Purpose:** Immediately identifies what the product is or does

**Guidelines:**
- Use common, searchable terms
- Be specific but concise
- Avoid jargon unless industry-standard

**Good Examples:**
- Phone Stand
- Cable Organizer
- Pen Holder
- Vase
- Cookie Cutter
- Keychain

**Bad Examples:**
- Thing-a-majig (too vague)
- Cylindrical Vertical Storage Solution (too complex)
- Awesome Holder (not descriptive)

---

### 2. Distinctive Feature
**Purpose:** Differentiates this product from similar items

**Types of Features:**
1. **Functional:** Adjustable, Collapsible, Modular, Stackable
2. **Design:** Geometric, Organic, Minimalist, Vintage, Modern
3. **Capacity:** 5-Slot, 10-Piece, Dual-Compartment
4. **Compatibility:** iPhone 15, Universal, MagSafe
5. **Style:** Hexagonal, Spiral, Lattice, Steampunk

**Examples:**
- Phone Stand - **Adjustable Angle**
- Dice Tower - **Dragon Design**
- Pen Holder - **6 Compartment**
- Headphone Stand - **RGB Backlit**

---

### 3. Material Specification
**Purpose:** Clearly indicates what material is used

**Format:** Use standard material codes from SKU_SYSTEM.md

**Examples:**
- PLA
- PLA+
- PETG
- TPU (Flexible)
- ABS
- Resin
- Wood-Fill PLA
- Carbon Fiber Nylon

---

### 4. Size/Variant Indicator
**Purpose:** Specifies size or variant when multiple options exist

**Format:** Always in parentheses at the end

**Size Examples:**
- (Small)
- (Medium)
- (Large)
- (XL)
- (100mm x 50mm)

**Variant Examples:**
- (Set of 4)
- (Single Unit)
- (Left-Handed)
- (Version 2.0)

## Category-Specific Guidelines

### HOME & LIVING
**Focus on:** Function + Style + Room context

**Template:** `[Item Type] - [Design Style/Feature] - [Material] ([Size])`

**Examples:**
- Planter - Minimalist Geometric - PLA (Medium)
- Soap Dish - Drainage Grid - PETG (Standard)
- Coat Hook - Modern Dual - PLA+ (Large)
- Utensil Holder - Rotating 4-Section - PLA (Large)

---

### TOYS & GAMES
**Focus on:** Type + Character/Theme + Play value

**Template:** `[Toy Type] - [Character/Theme] - [Material] ([Size/Age])`

**Examples:**
- Action Figure - Dragon Warrior - PLA (15cm)
- Puzzle Cube - Infinity Maze - PLA+ (Medium)
- Fidget Spinner - Galaxy Design - PETG (Standard)
- Dice Set - Dungeon Master - Resin (Standard D&D)

---

### TECH & GADGETS
**Focus on:** Device compatibility + Function + Features

**Template:** `[Device] [Accessory Type] - [Feature/Compatibility] - [Material] ([Size])`

**Examples:**
- Phone Stand - MagSafe Compatible - PLA (Universal)
- Cable Organizer - 6-Channel Desktop - PETG (Medium)
- Headphone Holder - RGB LED Base - PLA+ (Standard)
- Monitor Riser - Dual Screen - PLA (Large)

---

### AUTOMOTIVE
**Focus on:** Vehicle application + Function + Fitment

**Template:** `[Application] [Item Type] - [Feature/Fitment] - [Material] ([Compatibility])`

**Examples:**
- Cup Holder Insert - Dual Phone Mount - PETG (Universal)
- Door Panel Clip - OEM Replacement - ABS (Honda Civic 2020-2024)
- Dashboard Mount - Adjustable Phone - TPU (Universal)
- Console Organizer - Coin & Card - ABS (Tesla Model 3)

---

### JEWELRY & FASHION
**Focus on:** Item type + Design aesthetic + Wearability

**Template:** `[Jewelry Type] - [Design Description] - [Material] ([Size/Fit])`

**Examples:**
- Ring - Geometric Hexagon Band - PLA+ (Size 8)
- Earrings - Minimalist Drop - PETG (Lightweight)
- Pendant - Celtic Knot - Wood-Fill PLA (35mm)
- Brooch - Art Deco Fan - PLA (Medium)

---

### OFFICE & STATIONERY
**Focus on:** Function + Capacity + Organization

**Template:** `[Item Type] - [Feature/Capacity] - [Material] ([Size])`

**Examples:**
- Pen Holder - 8 Slot Hexagonal - PLA (Desktop)
- Business Card Holder - Tiered Display - PETG (50 Cards)

### MATERIALS & FILAMENTS
**Focus on:** Material + Quality tier + Color + Quantity

**Template:** `[Material Type] Filament - [Quality/Brand] - [Color] - [Weight]`

**Examples:**
- PLA+ Filament - Premium Plus - Silk Gold - 1kg

## Special Cases

### Limited Editions
Add "Limited Edition" prefix:
```
Limited Edition - [Product Name]

Example: Limited Edition - Dragon Dice Tower - Metallic Gold - Resin
```

### Seasonal Products
Include season/holiday:
```
[Holiday/Season] [Product Name]

Examples:
- Halloween - Pumpkin Planter - Orange PLA
- Christmas - Snowflake Ornament Set - White PLA (Set of 6)
```

### Collaborations
Include collaborator name:
```
[Product Name] - [Collaborator] Edition

Example: Geometric Vase - DesignerXYZ Edition - PLA+
```

### Beta/Prototype Products
Add version indicator:
```
[Product Name] - Beta v[X.X]

Example: Modular Organizer System - Beta v2.1 - PLA
```

## SEO & Searchability Best Practices

### 1. Keywords to Include
- Material type (PLA, PETG, etc.)
- Primary function (holder, stand, organizer)
- Application (phone, desk, kitchen)
- Style descriptors (modern, minimalist, geometric)

### 2. Avoid
- Special characters (!, @, #) except hyphens and parentheses
- ALL CAPS (except material codes)
- Excessive adjectives ("Amazing Super Cool")
- Trademarked terms without permission

### 3. Character Limits
- **Full Product Name:** 80 characters maximum
- **Short Name (for displays):** 40 characters maximum
- **Search Tags:** Create from name components

## Naming Checklist

Before finalizing a product name, verify:
- [ ] Clearly describes the product's primary function
- [ ] Includes distinctive feature(s)
- [ ] Specifies material
- [ ] Indicates size/variant if applicable
- [ ] Follows category-specific guidelines
- [ ] Is searchable and SEO-friendly
- [ ] Matches brand tone (professional, friendly)
- [ ] No spelling or grammar errors
- [ ] Within 80 character limit
- [ ] Unique in catalog (no duplicates)

## Name vs. SKU vs. Description

**Product Name:** Customer-facing, descriptive, searchable
```
Phone Stand - Adjustable Angle - PLA (Medium)
```

**SKU:** Internal tracking, systematic
```
PG-PDTC01-PLA-BLK-M-0001
```

**Short Description:** 1-2 sentences, marketing-focused
```
Ergonomic phone stand with adjustable viewing angles.
3D printed in durable PLA for desktop or bedside use.
```

**Full Description:** Detailed specifications, features, benefits
```
Our Adjustable Angle Phone Stand is designed for optimal viewing
comfort. Features smooth angle adjustment from 30-70 degrees,
non-slip base, and cable management slot. Compatible with all
smartphones. 3D printed in high-quality PLA for durability and
eco-friendliness. Available in multiple colors.

Dimensions: 120mm x 80mm x 60mm
Weight: 85g
Material: PLA (Polylactic Acid)
Print Resolution: 0.2mm layer height
Compatibility: Phones 4"-7" width
```
